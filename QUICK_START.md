# ⚡ Quick Start - EXE 빌드

## 단 3단계로 EXE 만들기!

### 1️⃣ 빌드 스크립트 실행
```cmd
build_exe.bat
```

### 2️⃣ 기다리기
- 가상환경 생성 중...
- 패키지 설치 중...
- EXE 빌드 중...
- ☕ 커피 한 잔 (5-10분 소요)

### 3️⃣ 완료!
```
✅ dist/MathHelper.exe 생성됨
```

---

## 🎮 사용 방법

### EXE 실행
```
MathHelper.exe 더블클릭
    ↓
자동으로 브라우저 열림
    ↓
앱 사용!
```

### 종료
- 검은 콘솔 창 닫기

---

## ❓ 문제 해결

### 빌드 안 됨
```cmd
pip install pyinstaller
```

### 실행 안 됨
- 백신 프로그램에서 허용

### 브라우저 안 열림
- 수동으로 열기: `http://localhost:8501`
