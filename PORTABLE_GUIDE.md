# 🎁 Portable 패키지 만들기 가이드

## 목표
**아들 PC에 Python 설치 없이, 그냥 실행만 되게!**

---

## 🚀 만드는 방법 (아버지 PC에서)

### 단 1줄:
```cmd
create_portable.bat
```

### 자동으로:
1. ✅ Python Embedded 다운로드 (30MB)
2. ✅ pip 설치
3. ✅ Streamlit, Plotly 등 패키지 설치
4. ✅ 앱 파일 복사
5. ✅ 실행 파일 생성
6. ✅ ZIP으로 압축

**소요시간:** 5-10분

---

## 📦 결과물

### 생성되는 것:
```
MathHelper_Portable/          (폴더)
├── 실행.bat                  ← 이것만 더블클릭!
├── 사용설명서.txt
├── python/                   (Python Embedded 포함)
├── src/                      (앱 소스코드)
└── config/                   (설정 파일)
```

### 또는:
```
MathHelper_Portable.zip       (ZIP 파일)
```

**크기:** 약 150-250MB

---

## 🎮 사용 방법 (아들 PC에서)

### 방법 1: 폴더로 전달
1. `MathHelper_Portable` 폴더를 USB에 복사
2. 아들 PC로 가져가기
3. 원하는 위치에 복사
4. `실행.bat` 더블클릭
5. 끝!

### 방법 2: ZIP으로 전달
1. `MathHelper_Portable.zip` 이메일/USB로 전달
2. 아들이 압축 해제
3. `실행.bat` 더블클릭
4. 끝!

## 🔍 상세 과정

### 1단계: Portable 패키지 생성 (아버지 PC)
```cmd
create_portable.bat
```

**진행 과정:**
```
📥 Python Embedded 다운로드 (30MB)
    ↓
📦 압축 해제
    ↓
🔧 pip 설치
    ↓
📦 패키지 설치 (Streamlit, Plotly 등)
    ↓
📋 앱 파일 복사
    ↓
```

### 2단계: 전달 (아버지 → 아들)

- 또는 이메일/클라우드

### 3단계: 실행 (아들 PC)
```
실행.bat 더블클릭
    ↓
브라우저 자동 열림
    ↓
사용!
```

## ⚠️ 주의사항

### 아들 PC 요구사항
- ✅ Windows 7 이상
- ✅ 인터넷: **불필요** (완전 오프라인 실행)
- ❌ Python: 불필요
- ❌ 관리자 권한: 불필요

### 첫 실행
- 백신 프로그램에서 경고 나올 수 있음
- "허용" 또는 "예외 추가" 선택

### 포트 충돌
- 다른 프로그램이 8501 포트 사용 시
- 해당 프로그램 종료 후 재시도

---

## 🛠️ 문제 해결

### "Python을 찾을 수 없음" 오류
→ 불가능! Portable 패키지는 Python 내장

### 브라우저 안 열림
→ 수동으로 열기: `http://localhost:8501`

### 실행 안 됨
1. 백신 프로그램 확인
2. `실행.bat` 우클릭 → 관리자 권한으로 실행

### ZIP 압축 해제 오류
→ Windows 기본 압축 프로그램 사용 (우클릭 → 압축 풀기)
