# MathHelper v1.0.19 사용자 테스트 가이드

## 🎯 테스트 목적
v1.0.19의 3가지 핵심 기능이 정상 작동하는지 검증

---

## ✅ 체크리스트

### 1. 기본 동작 (프로덕션 빌드 확인)

- [ ] **앱 실행 시 DevTools 자동으로 열리지 않음**
  - 실행 방법: MathHelper-v1.0.19-Windows-Portable.exe 더블클릭
  - 예상: 앱만 열리고 개발자 도구 없음

- [ ] **F12 눌러도 DevTools 안 열림**
  - 실행: F12 키 누르기
  - 예상: 아무 반응 없음

- [ ] **Ctrl+Shift+I 눌러도 DevTools 안 열림**
  - 실행: Ctrl+Shift+I 키 누르기
  - 예상: 아무 반응 없음

---

### 2. 카테고리 필터링

#### 2-1. 일차방정식 카테고리
- [ ] **좌측 메뉴에서 "일차방정식" 선택**
- [ ] **4개 모드만 표시 확인**:
  - ✅ 계산
  - ✅ 방정식 풀이
  - ✅ 간단히
  - ✅ 통합 계산
  - ❌ 미분 (없어야 함)
  - ❌ 적분 (없어야 함)
  - ❌ 인수분해 (없어야 함)
  - ❌ 전개 (없어야 함)
  - ❌ 극한 (없어야 함)

#### 2-2. 미분 카테고리
- [ ] **좌측 메뉴에서 "미분" 선택**
- [ ] **3개 모드만 표시 확인**:
  - ✅ 미분
  - ✅ 간단히
  - ✅ 계산
  - ✅ 통합 계산

---

### 3. 자동 모드 전환

#### 3-1. 방정식 자동 전환
- [ ] **"계산" 모드 선택**
- [ ] **입력창에 "2x + 4 = 10" 입력**
- [ ] **자동으로 "방정식 풀이" 모드로 전환됨**
- [ ] **토스트 알림 표시**: "📐 방정식 풀이 모드로 자동 전환됨"
  - 위치: 우측 하단
  - 지속시간: 3초 후 자동 숨김
- [ ] **상단 배지 표시**: "⚡ 📐 방정식 풀이 자동 적용"
  - 색상: 녹색 배경
  - 애니메이션: pulse 효과

#### 3-2. 계산 실행
- [ ] **"계산하기" 버튼 클릭**
- [ ] **에러 없이 결과 표시**: "x = 3"

---

### 4. 스마트 결과 요약

#### 4-1. 통합 계산 실행
- [ ] **입력창에 "4/3 * pi * 6^3" 입력**
- [ ] **"⚡ 통합 계산" 버튼 클릭**

#### 4-2. 통계 배너 확인
- [ ] **통계 배너 표시**:
  - ⚡ 통합 계산 완료
  - ✓ 성공 X개
  - ✗ 실패 X개
  - 🕐 XX.Xms

#### 4-3. 주 결과 확인
- [ ] **주 결과 크게 강조 표시**:
  - 배경: 그라데이션 (파란색 → 보라색)
  - 폰트 크기: 매우 큼 (2.5rem 이상)
  - 내용: "904.78" 또는 유사한 값
  - 라벨: "🎯 최종 답"
  - 모드 표시: "🧮 계산 | X.Xms"

#### 4-4. 추가 정보 확인
- [ ] **"▼ 추가 정보 (X개)" 버튼 존재**
- [ ] **버튼 클릭 시 접기/펼치기 동작**
- [ ] **추가 정보 카드 표시**:
  - 간단히: 904.78
  - 전개: 904.78

#### 4-5. 적용 불가 확인
- [ ] **"▼ 적용 불가 (X개)" 버튼 존재**
- [ ] **버튼 클릭 시 펼치기**
- [ ] **실패 이유 명확히 표시**:
  - ✗ 방정식 풀이: 등호가 없어 방정식을 풀 수 없습니다
  - ✗ 미분: 변수가 없어 미분할 수 없습니다
  - ✗ 적분: 변수가 없어 적분할 수 없습니다

#### 4-6. 결과 카드 클릭
- [ ] **추가 정보 카드 중 하나 클릭**
- [ ] **해당 모드로 전환됨**
- [ ] **해당 결과가 단일 결과로 표시됨**

---

### 5. 에러 처리

#### 5-1. 잘못된 입력
- [ ] **입력창에 "2x +" 입력 (불완전한 수식)**
- [ ] **"계산하기" 클릭**
- [ ] **사용자 친화적 에러 메시지 표시**:
  - ⚠️ 입력 중 오류가 발견되었습니다.
  - 💡 괄호가 올바르게 닫혔는지, 연산자가 누락되지 않았는지 확인해주세요.
- [ ] **기술 정보는 "접기" 상태로 숨겨짐**
- [ ] **"기술 정보 보기" 클릭 시 펼쳐짐**

#### 5-2. 이중 부정 메시지 없음
- [ ] **모든 에러 메시지에 "사용할 수 없습니다" 문구 없음**
- [ ] **대신 "선택해주세요", "확인해주세요" 등 긍정 문구 사용**
