## ✨ 새로운 기능 (v1.0.19)

### 🎯 1. 스마트 결과 요약
통합 계산 시 **최종 답을 크게 강조** 표시하여 한눈에 파악 가능!

**Before (v1.0.18)**:
```
[계산: 904.78] [방정식: 실패] [미분: 실패] [적분: 실패]
[간단히: 904.78] [인수분해: 실패] [전개: 904.78] [극한: 실패]
→ 어떤 게 최종 답인지 불명확 ❌
```

**After (v1.0.19)**:
```
┌─────────────────────────────────────┐
│ 🎯 최종 답                          │
│                                     │
│     904.78                          │
│                                     │
│ 🧮 계산 | 0.3ms                     │
└─────────────────────────────────────┘

▼ 추가 정보 (2개)
  • 간단히: 904.78
  • 전개: 904.78

▼ 적용 불가 (5개)
  ✗ 방정식 풀이: 등호가 없어 방정식을 풀 수 없습니다
→ 최종 답 즉시 파악 가능! ✅
```

### 🔄 2. 자동 모드 전환
방정식을 입력하면 **자동으로 방정식 모드**로 전환됩니다.

**Before (v1.0.18)**:
```
입력: 2x + 4 = 10 (계산 모드)
❌ 에러: 등호(=)는 방정식 모드에서만 사용할 수 없습니다
→ 사용자가 수동으로 모드 변경
→ 다시 계산 버튼 클릭
```

**After (v1.0.19)**:
```
입력: 2x + 4 = 10 (계산 모드)
🔄 토스트: "방정식 풀이 모드로 자동 전환됨"
✅ 배지: "⚡ 방정식 풀이 자동 적용"
✅ 결과: x = 3
→ 에러 없이 바로 계산 완료!
```

### 📁 3. 카테고리 필터링
일차방정식 카테고리에서는 **관련 계산기만** 표시됩니다.

- 일차방정식: 방정식 풀이, 계산, 간단히, 전개 (4개)
- 이차방정식: 방정식 풀이, 인수분해, 계산, 전개 (4개)
- 미분: 미분, 간단히, 계산 (3개)

→ 통합 계산 성공률 **87.5%**로 향상!

## 주요 기능

### 계산 모드 (8가지)
- 🧮 **계산**: 수식 계산
- 📐 **방정식**: 방정식 풀이
- ∂ **미분**: 함수 미분
- ∫ **적분**: 함수 적분
- ✨ **간단히**: 식 정리
- 🔨 **인수분해**: 인수분해
- 📦 **전개**: 식 전개
- ∞ **극한**: 극한값 계산

### 키보드 단축키 ⌨️
- `Ctrl+1~8`: 모드 빠른 전환
- `Ctrl+Enter`: 계산 실행
- `Ctrl+K`: 가상 키보드 토글

### 국제화 🌐
- 한국어 / English 지원
- localStorage에 언어 설정 저장

### 계산 히스토리 📜
- 계산 결과 자동 저장 (최대 100개)
- 클릭으로 이전 계산 복원
- 모드별 필터링
- 개별/전체 삭제
- 상대 시간 표시 (방금 전, N분 전 등)

## 기술 스택

- **Frontend**: React 18 + TypeScript + Vite
- **UI**: TailwindCSS + Lucide Icons
- **Math Engine**: mathjs (계산) + nerdamer (기호 연산)
- **Desktop**: Electron 28
- **i18n**: i18next + react-i18next
- **Testing**: Vitest (Unit) + Playwright (E2E)
